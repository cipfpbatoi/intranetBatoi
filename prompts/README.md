# Prompts reutilitzables

Plantilles per a tasques que es repeteixen. **Regla de les 3 vegades**: si has d'explicar el mateix en >3 prompts, crea o actualitza una plantilla ací.

## Com usar-les

1. Copia el fitxer de plantilla corresponent.
2. Substitueix els marcadors `{{...}}`.
3. Envia-la a l'agent directament — el context necessari ja hi és.

## Índex

| Fitxer | Tasca |
|---|---|
| [`nou-controlador.md`](nou-controlador.md) | Nou controlador (BaseController CRUD o personalitzat) |
| [`nova-entitat.md`](nova-entitat.md) | Nova entitat Eloquent + Policy + migració |
| [`nou-test-feature.md`](nou-test-feature.md) | Nou test Feature PHPUnit |
| [`nova-vista-blade.md`](nova-vista-blade.md) | Nova vista Blade amb layout i parcials |
| [`refactor-a-service.md`](refactor-a-service.md) | Extreure lògica d'un controlador a Application/Service |
| [`review-checklist.md`](review-checklist.md) | Revisió de codi per un agent revisor independent |

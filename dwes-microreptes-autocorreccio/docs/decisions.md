# Decisions

| Decisio | Estat | Motiu |
| --- | --- | --- |
| Repositori centralitzat | Acceptada | Manté microreptes, rúbriques i polítiques en una font única. |
| Microrepte actiu controlat pel professor | Acceptada | Evita modificar repositoris d'alumnes per canvis de calendari. |
| Nota provisional automàtica | Acceptada | L'autocorrecció orienta però no substitueix sempre la revisió docent. |
| Revisió docent amb baixa confiança o flags | Acceptada | Protegeix l'avaluació davant evidències insuficients. |
